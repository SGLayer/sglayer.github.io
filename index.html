SGL (Simple GPU Layer), a high-performance GPU compute and graphics framework built on Vulkan. SGL is designed to enable efficient GPU-accelerated computation and rendering across multiple platforms, providing both low-level compute primitives and high-level processing engines for image manipulation, matrix operations, and 3D visualization.
